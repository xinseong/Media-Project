# MediaProject
# 📱 몇 ML: AR 기반 비정형 용기 부피 측정 서비스
> [cite_start]**아주대학교 미디어프로젝트(M024) 캡스톤디자인** [cite: 124]
> **"계량컵 없는 세상, 카메라만으로 정확하게 액체 부피를 측정하다."**

[cite_start]`몇 ML`은 1인 가구의 증가와 요리 레시피 계량의 불편함에서 착안하여 [cite: 124][cite_start], 별도의 물리적 도구 없이 스마트폰 카메라와 AI 추론 서버 연동을 통해 복잡한 비정형 용기 내부의 액체 용량($ml$)을 정밀하게 측정하는 AR 기반 솔루션입니다[cite: 124].

---

## 💡 Project Overview (프로젝트 개요)

* [cite_start]**추진 배경:** 가정마다 계량 도구의 크기가 다르고 구비되지 않은 경우가 많아 정확한 레시피 재료 계량에 어려움을 겪습니다[cite: 124].
* [cite_start]**솔루션:** 스마트폰 카메라로 용기를 촬영하면 시스템이 용기의 기하학적 형태와 수면선 높이를 실시간 분석하여 액체 부피를 자동 계산합니다. 
* [cite_start]**핵심 가치:** 광학적 측정 방식으로 교차 오염을 방지하여 위생적이며 [cite: 125][cite_start], AR 목표 수위선 렌더링을 통해 직관적인 UX를 제공합니다[cite: 125].
